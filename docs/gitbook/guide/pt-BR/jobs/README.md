# Jobs

As filas podem conter diferentes tipos de *jobs* que determinam como e quando serão processados. Nesta seção, iremos descrevê-los em detalhes.

Uma coisa importante a se considerar é que você pode misturar os diferentes tipos de *jobs* na mesma fila, de modo que pode adicionar trabalhos FIFO e, a qualquer momento, adicionar um LIFO ou um *delayed job*.
